# adreport

Отчёты по постам Telegram-каналов: аккуратный PDF и PNG-карточка вместо
скриншотов из статистики. Сервис канал-агностик: имя, контакт и таймзона —
параметры конфига, тематика нигде не зашита.

**Статус: v0.1** — отчёт из фикстур. Телеграма в этой версии ещё нет:
`adreport demo` собирает витринный отчёт без токенов и сессий.
Примеры готовых файлов — в [examples/](examples/).

## Быстрый старт

```bash
python -m venv .venv
.venv/Scripts/pip install -e ".[dev]"   # Linux/macOS: .venv/bin/pip
adreport demo            # PDF + PNG из фикстуры полной статистики → out/
adreport demo --public   # вариант «только публичные данные»
adreport demo --pdf      # только PDF
```

### WeasyPrint на Windows

WeasyPrint нужны нативные библиотеки Pango/GObject, которые pip не ставит.
Без установки в систему: скачать `GTK3_Gvsbuild_*_x64.zip` из
[релизов gvsbuild](https://github.com/wingtk/gvsbuild/releases) и распаковать
`bin/` (и, желательно, `etc/fonts/`) в `.gtk3/` в корне репозитория — рендерер
подхватит DLL сам. Альтернативы: переменная `ADREPORT_GTK_BIN` с путём к DLL
или MSYS2 (см. документацию WeasyPrint). На Linux/macOS достаточно пакетов
`pango`/`libpango` из системного менеджера.

## Тесты

```bash
.venv/Scripts/python -m pytest
```

Golden-тест рендерит фикстуру с «Ёё Щщ №» и проверяет, что кириллица и эмодзи
набраны вшитыми шрифтами (IBM Plex Sans, Noto Emoji — OFL), а не системным
фолбэком: квадраты ловятся автоматически.
